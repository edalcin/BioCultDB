## Remediação de segurança

1. **Credencial MongoDB vazada**: `.claude/settings.local.json` (rastreado no git desde 2025-12-31, sem estar no `.gitignore`) continha uma URI real de conexão com usuário, senha e IP de produção em texto plano. A senha foi **rotacionada no servidor MongoDB** pelo usuário. O arquivo foi removido do tracking do git (`git rm --cached`) e adicionado ao `.gitignore` — permanece no disco local, mas não será mais versionado. A string com a senha antiga permanece no histórico do git (decisão consciente de não reescrever o histórico), porém está inofensiva desde a rotação.
2. **Agente `.claude/agents/security-reviewer.md`**: descrevia o projeto errado ("etnotermos", porta 4001 — provável cópia de um repositório irmão). Corrigido para descrever corretamente o BioCultDB (3 contextos, portas 3001-3003).
